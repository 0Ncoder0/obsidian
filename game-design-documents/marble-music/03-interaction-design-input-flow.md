# 03 -- 交互设计与输入映射

## 1. 输入设备与前提

- **输入设备**：鼠标 + 键盘（PC Web 环境）
- **核心原则**：所有输入的行为解释取决于当前模式（编辑 / 播放）。同一操作在不同模式下可能有不同响应或被完全忽略。

---

## 2. 全局输入映射（两种模式共用）

以下操作在编辑态和播放态中均生效：

| 输入                    | 行为           | 说明                                     |
| ----------------------- | -------------- | ---------------------------------------- |
| 鼠标滚轮               | 缩放视窗       | 以鼠标位置为中心缩放                     |
| 中键拖拽               | 平移视窗       | 拖拽移动画布可视区域                     |
| `Alt` + 左键拖拽        | 平移视窗       | 中键拖拽的替代方案（无中键鼠标适用）     |
| `Space`                 | 模式切换       | 编辑 -> 播放 / 播放 -> 编辑              |
| `Esc`                   | 上下文退出     | 播放态：停止播放。编辑态：取消临时放置态 |

---

## 3. 编辑模式输入映射

以下操作仅在编辑模式下生效。播放态中这些操作要么被忽略，要么给出轻提示。

### 积木操作

| 输入                      | 行为               | 说明                                    |
| ------------------------- | ------------------ | --------------------------------------- |
| 左键点击右侧积木按钮      | 选择放置类型       | 切换当前要放置的积木类型                |
| `1`                       | 快捷选择：小球     | 等效于点击积木按钮"小球"                |
| `2`                       | 快捷选择：方块     | 等效于点击积木按钮"方块"                |
| `3`                       | 快捷选择：音乐方块 | 等效于点击积木按钮"音乐方块"            |
| 左键点击画布空白区域      | 放置积木           | 在点击位置放置当前选中类型的积木        |
| 左键点击画布上的实体      | 选中实体           | 高亮该实体，右侧面板显示其属性          |
| 左键拖拽画布上的实体      | 移动实体           | 改变实体在场景中的位置                  |
| `Delete` / `Backspace`   | 删除选中实体       | 删除当前选中的实体，清空选中状态        |

### 参数编辑

| 输入                     | 行为               | 说明                                    |
| ------------------------ | ------------------ | --------------------------------------- |
| 选中音乐方块后编辑面板    | 修改音乐参数       | 可调整 `noteName`、`volume`、`durationMs` |

参数变更实时生效并自动进入保存队列。

---

## 4. 播放模式输入映射

播放模式下仅允许视窗操作和停止播放：

| 输入         | 行为       | 说明                               |
| ------------ | ---------- | ---------------------------------- |
| `Space`      | 停止播放   | 返回编辑模式                       |
| `Esc`        | 停止播放   | 返回编辑模式                       |
| 鼠标滚轮     | 缩放视窗   | 正常工作                           |
| 中键拖拽     | 平移视窗   | 正常工作                           |
| 其他编辑操作 | **被阻断** | 不产生任何效果，可选择性给出轻提示  |

---

## 5. 多球选中与跟随交互（定稿）

### 选中行为

1. 场景中有多个小球时，点击某个小球仅表示"选中"该球
2. 选中不等于跟随 -- 选中是编辑态的行为，跟随是播放态的行为
3. 同一时刻最多只有一个选中球
4. 点击空白区域或其他非球实体不会自动清空选中球

### 跟随判定（播放启动时）

```
用户按下 Space（编辑态）
         │
         ├─ 存在选中球？
         │    ├─ Yes: 播放开始后，相机自动跟随该球
         │    └─ No:  播放开始后，相机保持当前位置
         │
         ▼
   进入播放模式
```

### 播放期间的相机行为

- 自动跟随开启时：相机平滑追踪选中球，但用户可随时手动平移/缩放覆盖
- 无自动跟随时：相机完全由用户手动控制
- 播放中不支持切换跟随目标

### 状态保留

- 停止播放返回编辑态后，选中球状态保留
- 直到用户显式点击另一个小球（改选）或点击非球区域/实体

## 6. 音乐方块参数编辑交互

### 触发条件

在编辑模式下选中音乐方块时，右侧面板显示参数编辑区。

### 参数字段

| 字段         | 输入类型     | 约束                  | 说明               |
| ------------ | ------------ | --------------------- | ------------------ |
| `noteName`   | 文本 / 选择器 | 有效音名（如 `C4`）  | 决定发声音高       |
| `volume`     | 滑块 / 数字  | 0 ~ 1                | 触发初始音量       |
| `durationMs` | 数字输入     | 正整数，> 0           | 包络持续时长（ms） |

### 编辑行为

- 参数修改实时生效，无需"确认"按钮
- 修改后自动进入保存队列（节流写入）
- 非法输入（如负数、空音名）即时提示，不写入非法值
- 播放模式下参数面板不可见（随右侧面板整体隐藏）

## 7. 模式切换完整流程

### 编辑 -> 播放

```
1. 捕获 Space 键按下
2. 锁定所有编辑输入（新增 / 移动 / 删除 / 参数修改）
3. 隐藏右侧积木选择面板和参数编辑面板
4. 隐藏底部 Timeline UI
5. 隐藏预测线
6. 判定跟随目标：
   - 若 selectedBallId 存在且对应球有效 → 启用自动跟随
   - 否则 → 不启用自动跟随
7. 启动物理引擎（开始步进模拟）
8. 启动音频系统（准备接收碰撞触发）
9. 开始记录 ContactEvent 到 Timeline 数据流
```

### 播放 -> 编辑

```
1. 捕获 Space 或 Esc 键按下
2. 停止物理引擎步进
3. 停止音频触发（已发声的音符允许自然衰减完毕）
4. 恢复编辑输入权限
5. 显示右侧面板
6. 显示底部 Timeline（展示本轮播放的事件记录）
7. 恢复预测线显示
8. 触发一次强制保存（将当前场景状态写入 localStorage）
```

## 8. 反馈设计

| 场景           | 反馈方式                                             |
| -------------- | ---------------------------------------------------- |
| 实体被选中     | 高亮轮廓 + 右侧面板同步显示该实体属性               |
| 播放态尝试编辑 | 忽略操作（不执行），可选择给出轻量视觉/文字提示      |
| 自动保存成功   | 状态栏短暂显示"已保存"                               |
| 自动保存失败   | 状态栏持续显示"保存失败"并附可行动提示               |
| 积木类型切换   | 当前选中按钮高亮，光标样式变化                       |
| 音乐方块发声   | 音乐方块上显示脉冲/高亮动画，与音量同步              |
